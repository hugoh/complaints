Flight maps

If you view the map for a flight, you'll see a trail of colored dots. Each is a specific Trackpoint from a data source; click it to see details about that point.

Yellow dots: data source was Mode-ES location info broadcast from the aircraft itself, captured on a privately operated ADS-B receiver.
Blue dots: track data was purchased from FlightAware (radar or ADS-B/ES).
Big red dots: this particular point (from any source) was interpreted as a Class B violation. Click on it to learn why.
Green dots: data source was Flightradar24. Data is not used in any reporting, and is provided for just for comparison.

The ClassB detection logic uses a conservative/simplifed view of class B space that is only appropriate for flights flying in on SERFR1.
